Sensor Server Firmware for Espressif Microcontrollers

A full stack frontend + backend system for monitoring sensor readings from multiple ESP32 and ESP8266 microcontrollers.

Whether it’s for laboratory work, robotics, or just plain old DIY smart home magic, you’ll need a way to coordinate all your smart tools and appliances while conserving energy.

The Sensor Server is an open source firmware for ESP32 and ESP8266 microcontrollers. With the Sensor Cluster, it allows multiple microcontrollers to input information straight into a database warehouse with minimal setup.

This project was designed with large-distance field and laboratory monitoring in mind, meant for use within the thick forests of Nueva Vizcaya and the rice research paddies of the Science City of Munoz.

Check out the project at the following links:

ESP8266 GitHub Repository: https://github.com/Frostzeichen/Statistics-in-C
