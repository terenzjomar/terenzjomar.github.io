Statistics in C

An open source C library for running advanced AI and data computations on card-sized embedded computers.

Machine learning and big data crunch a huge amount of data all the time. The general rule of thumb, the bigger the data, the better the dataset holds true for most machine learning projects. And this does become a problem for edge-computed devices with roughly the same memory capability as a pocket calculator.

In comes Statistics in C. This is an ongoing project I am building to compute data right where it’s collected.

Imagine having a safer world where self-driving cars can quickly compute whether it’s safe to drive or park for maybe five seconds. Even in the middle of a snowstorm without internet. That’s the kind of world this library opens things up for, and more!

Check out the project at the following links:
